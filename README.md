# Microsoft-365-Enterprise-Support-Lab

## Présentation du projet

Ce projet simule un environnement Microsoft 365 d’entreprise afin de développer des compétences pratiques en Support IT N1/N2.

Le laboratoire reproduit des scénarios réels rencontrés dans un environnement professionnel Microsoft 365 :
- Administration des utilisateurs
- Gestion des licences
- Microsoft Teams
- SharePoint Online
- Exchange Online
- Microsoft Entra ID
- Gestion des permissions
- Résolution d’incidents utilisateurs

L’objectif est de documenter la configuration, les incidents rencontrés et les solutions appliquées dans une logique de troubleshooting professionnel.

---

# Technologies utilisées

- Microsoft 365 Admin Center
- Microsoft Entra ID
- Microsoft Teams
- SharePoint Online
- Exchange Online
- Outlook
- OneDrive

---

# Objectifs du laboratoire

- Créer et administrer des utilisateurs Microsoft 365
- Configurer des équipes Teams
- Gérer les permissions SharePoint
- Tester les licences Microsoft 365
- Simuler des incidents Support IT
- Documenter les diagnostics et solutions
