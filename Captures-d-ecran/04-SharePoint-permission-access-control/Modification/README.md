# SharePoint – Permission Modification

## Objectif
Permettre à un utilisateur de modifier un document SharePoint.

## Actions réalisées

- Activation des autorisations uniques
- Partage du document Contrats-RH
- Attribution du rôle Modification à temp.user
- Validation via Outlook

## Résultat

L’utilisateur peut :
- modifier le document
- enregistrer les changements
- collaborer sur le fichier
