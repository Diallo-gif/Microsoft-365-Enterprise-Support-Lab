# SharePoint – Accès restreint

## Objectif
Limiter l’accès à un document sensible sans autoriser la modification, le téléchargement ou l’impression.

## Actions réalisées

- Ajout de temp.user comme visiteur du site Finance
- Configuration de permissions uniques sur le document Factures
- Attribution du niveau Affichage restreint
- Validation avec le compte utilisateur

## Résultat

L’utilisateur peut :
- consulter le document

L’utilisateur ne peut pas :
- modifier
- télécharger
- imprimer
