## Scénario – Accès lecture seule

Objectif :
Permettre à un utilisateur invité de consulter un document SharePoint sans pouvoir le modifier.

### Étapes réalisées

1. Ajout de l’utilisateur guest.partner dans l’équipe HR Département
2. Ouverture du site SharePoint depuis Teams
3. Configuration des autorisations du document
4. Attribution du rôle Lecture
5. Connexion avec le compte invité pour validation

### Résultat

L’utilisateur invité peut :
- ouvrir le document
- lire son contenu

L’utilisateur ne peut pas :
- modifier
- supprimer
- partager le fichier
